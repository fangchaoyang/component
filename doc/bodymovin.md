```
bodamovin是Adobe After Effects中的插件，可以将AE动画导出成为json文件并在各种设备上呈现。
```
```
使用方法：
打开AE项目并在Window> Extensions> bodymovin上选择bodymovin扩展
面板打开，其中包含列出所有项目组合的“组合”选项卡。
选择要导出的合成。
选择目标文件夹。
单击“渲染”
查找导出的json文件（如果动画中有图像或AI图层，则会有一个包含导出文件的图像文件夹）
```
```
注意事项：
该脚本支持预编译，形状，实体，图像，空对象，文本
它支持掩模和倒置掩模。也许其他模式会出现，但它会有巨大的性能影响。
它支持时间重映射
该脚本支持形状，矩形，椭圆和星形。
不支持：不支持图像序列，视频和音频
没有负面层拉伸！不知道为什么，但伸展一层混乱的所有数据
```
